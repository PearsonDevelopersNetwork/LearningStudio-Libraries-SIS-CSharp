ClientInspector
Custom IClientMessageInspector class used to override the default WCF Message behavior. This is required for interoperability between a WCF Client and a WSE 2.0 Service that implements UsernameToken security using a PasswordDiget.
InspectorBehavior
Custom IEndpointBehavior class used to override the default WCF endpoint behavior. This is required for interoperability between a WCF Client and a WSE 2.0 Service that implements UsernameToken security using a PasswordDiget.
SecurityHeader
Custom MessageHeader class used to override the default WCF generated SOAP Security Header. This is required for interoperability between a WCF Client and a WSE 2.0 Service that implements UsernameToken security using a PasswordDiget.
